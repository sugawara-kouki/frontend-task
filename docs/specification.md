# 仕様書

## 概要
サイドバーナビゲーションとメインコンテンツエリアを持つ、シンプルなページ管理CRUDアプリケーション。

## 機能要件

### 1. サイドバーナビゲーション
- サイドバーに全ページを表示する
- 既存ページのリストを表示し、簡単にナビゲーションできる

### 2. ページ作成
- メニュー下部の「+」ボタンをクリックしてページを追加する
- 各ページはデフォルトで空のタイトルと本文で作成される

### 3. ページ削除
- 「-」ボタンをクリックしてページを削除する
- サイドバーとストレージからページを削除する

### 4. ページ編集
- 各ページは以下で構成される：
  - **タイトル**: 編集可能なテキストフィールド
  - **本文**: 編集可能なテキストエリア
- 変更を永続化するための保存機能

## バリデーションルール

### タイトルのバリデーション
- **最小文字数**: 1文字
- **最大文字数**: 50文字
- 保存時に空であってはならない

### 本文のバリデーション
- **最小文字数**: 10文字
- **最大文字数**: 2000文字
- 保存時に最小文字数を満たす必要がある

## テスト要件
- 有効だと思われるテストケースを実装する
- 最低1つのテストが必須
- テストカバレッジに含めるべき項目：
  - CRUD操作（作成、読み取り、更新、削除）
  - バリデーションルール
  - ユーザーインタラクション

## 追加機能
要件で指定されていない追加機能や改善がある場合は、適宜実装してください。追加した内容はREADME.mdファイルに記載してください。
